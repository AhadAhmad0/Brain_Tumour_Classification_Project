About the Project

This project uses a deep learning model to classify MRI images into two categories: No Brain Tumor and Yes Brain Tumor. The system is designed to provide fast and simple image-based prediction through a web interface built with Flask.

Developed using Flask, TensorFlow, OpenCV, and Deep Learning
